# Sales, Customer & Inventory Analytics (MySQL + Power BI)

## Overview
Designed and optimized a MySQL database consolidating 500K+ customer, sales, and inventory records to enable consistent reporting and analytics.

## Key Features
- Optimized relational database design (MySQL)
- Automated data generation for large-scale datasets
- KPI-driven analytics using SQL and Power BI
- Identified underperforming products and customer trends

## KPIs Tracked
- Sales Revenue
- Top & Bottom Performing Products
- Customer Segmentation
- Regional Sales Trends
- Inventory Stock Levels

## Tools Used
- MySQL
- Power BI
- SQL
- DAX

## Business Impact
- Improved data accuracy by 98%
- Identified 20% underperforming products
- Increased customer retention by 15%
- Improved sales efficiency by 20%
